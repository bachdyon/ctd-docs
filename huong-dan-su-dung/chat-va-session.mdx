## Luồng thường dùng

1. Tạo session mới.
2. Gửi request chat trong session.
3. Xem lịch sử chat theo session.
4. Chia sẻ session bằng share UUID (nếu được bật).
5. Fork session khi cần tạo nhánh làm việc mới.

## Mẹo sử dụng

- Đặt tên session rõ ràng theo mục tiêu nội dung.
- Dùng fork trước khi thử các thay đổi lớn để tránh mất ngữ cảnh cũ.
- Dùng replay/edit request để tinh chỉnh kết quả AI.
