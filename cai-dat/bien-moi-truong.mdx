Giá trị mặc định / tên biến lấy theo [`.env.example`](https://github.com/bachdyon/vtd/blob/main/.env.example) trên repo. **Không commit file `.env`** thật.

Cột **Bắt buộc / tùy chọn** hiển thị <strong>[Bắt buộc]</strong> hoặc <strong>[Tùy chọn]</strong>; dòng dưới là lý do (<small>chữ nhỏ</small>).

## Core

| Biến | Ý nghĩa | Bắt buộc / tùy chọn |
|------|---------|---------------------|
| `PORT` | Cổng HTTP của Flask API khi chạy `python -m api.main`. | <strong>[Tùy chọn]</strong><br /><small>Có giá trị mặc định trong code (ví dụ dev).</small> |
| `DEBUG_MODE` | `true`/`false` — chế độ debug API. | <strong>[Tùy chọn]</strong><br /><small>Prod nên `false`.</small> |

### Video agent — cấu trúc dưới GENERATED_FOLDER {#video-agent-generated-layout}

Code: `workers/videos/agent/file_manager.py`. Mỗi **job** có `job_id` (Mongo id dạng chuỗi). File tạm nằm tại:

```text
{GENERATED_FOLDER}/images/{job_id}/<file>
{GENERATED_FOLDER}/videos/{job_id}/<file>
```

Tham số `asset_type` trong code chỉ để chọn nhánh **images** (nhóm ảnh) hay **videos** (voice, transcript, video, …); **không** bắt buộc có thư mục con `{asset_type}/` — file nằm trực tiếp dưới `{job_id}/`.

Khi pipeline xong, `cleanup()` xóa cả `.../images/{job_id}` và `.../videos/{job_id}` nếu tồn tại.

## Lưu trữ object (MinIO / R2 / S3)

| Biến | Ý nghĩa | Bắt buộc / tùy chọn |
|------|---------|---------------------|
| `STORAGE` / `FILE_STORAGE_BACKEND` | `minio`, `r2`, `s3`. MinIO: biến `MINIO_*`. S3 (AWS hoặc endpoint tùy chỉnh): `S3_*` (`S3_ENDPOINT_URL` có thể để trống khi dùng AWS). | <strong>[Bắt buộc]</strong><br /><small>Mặc định dự án là `minio`. Không còn hỗ trợ backend local object storage.</small> |
| `CLOUDFLARE_R2_*` | Account, key, bucket, domain R2. Tùy chọn `CLOUDFLARE_R2_ADDRESSING_STYLE` = `path` hoặc `virtual` (boto3 S3), hiếm khi cần đổi so với mặc định. | <strong>[Bắt buộc]</strong><br /><small>Khi `STORAGE=r2`: ít nhất account, key, bucket hợp lệ.</small> |
| `MINIO_*` / `S3_*` | Cấu hình MinIO hoặc S3 khi `STORAGE=minio` / `s3`. | <strong>[Bắt buộc]</strong><br /><small>Đúng bộ biến theo từng chế độ (xem `.env.example`). Với MinIO nên đặt `MINIO_ADDRESSING_STYLE=path`.</small> |
| `MINIO_AUTO_CREATE_BUCKET` | Tự tạo bucket MinIO khi API khởi động nếu bucket chưa tồn tại. | <strong>[Tùy chọn]</strong><br /><small>Mặc định `true` trong code để không cần tạo bucket bằng tay ở môi trường local/dev.</small> |
| `MINIO_BUCKET_PUBLIC_READ` | Tự set policy public-read (`s3:GetObject`) cho bucket MinIO khi API khởi động. | <strong>[Tùy chọn]</strong><br /><small>Mặc định `true` để test nhanh. Trên production nên cân nhắc đặt `false` và dùng presigned URL.</small> |
| `MAX_FILE_SIZE` / `STORAGE_PRICE_PER_GB` | Giới hạn kích thước / hiển thị giá (nếu có). | <strong>[Tùy chọn]</strong> |
